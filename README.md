# Smart Recommendation System

The Smart Recommendation System is a simple Python application that suggests movies, books, and music based on user input. The system is built using Python and Tkinter to provide a basic graphical user interface.

This project demonstrates the concept of a content-based recommendation system, where suggestions are generated based on predefined data.

## Features
- Recommends Movies, Books, and Music
- Simple and user-friendly GUI
- Built using Python Tkinter
- Beginner-friendly project

## Technologies Used
- Python
- Tkinter

## Project Structure
movie-recommendation-system|
--recommendation.py
--README.md

## Description
The user selects a category such as Movie, Book, or Music and enters a name. The system searches its dataset and provides related recommendations if the item is found.

This project was developed as a beginner-level recommendation system to understand basic AI concepts and GUI development in Python.
